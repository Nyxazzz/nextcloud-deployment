# Seguridad

## HTTPS

Certificados TLS para aumentar el cifrado de la web y los datos de navegacion.

## Firewall

Puertos abiertos:
- 8080
- 80
- 443

## MFA

Autenticación multifactor para aumentar la seguridad en los usuarios que van a usar el servicio de nube.

## Actualizaciones
Posible automatización de actualización tanto del servicio en la nube como del propio sistema operativo del Servidor
