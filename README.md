# Nextcloud Deployment

Despliegue completo de Nextcloud utilizando Docker sobre Ubuntu Server

Incluye:

- Docker
- MariaDB
- HTTPS
- Firewall
- Backups
- Gestión de usuarios

# Problema

Muchas pequeñas empresas utilizan google Drive o almacenan archivos localmente sin control sobre sus datos.

Este proyecto proporciona una nube privada basada en el software libre NextCloud, que nos proporciona el almacenamiento que nosotros tengamos en el equipo donde lo instalaremos.

## Tecnologias utilizadas

- Ubuntu Server 24.04
- Docker
- Docker Compose
- Nextcloud
- MariaDB
- UFW
